# prospect-service-spring

## To install

mvn clean install

## To run

mvn spring-boot:run

## MongoDB

To install:

https://www.mongodb.com/

Optional:

https://robomongo.org/

To import:

`mongoimport --db mean --collection prospects --file prospects.json`
